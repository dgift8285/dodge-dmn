## 📋 Commands

### 🌐 General
| Command | Description |
|---|---|
| `.ping` | Check bot latency |
| `.alive` / `.online` | Check bot status |
| `.uptime` | Show bot uptime |
| `.owner` | Show linked number |
| `.menu` / `.help` | Show full command menu |

### ⚙️ Settings
| Command | Description |
|---|---|
| `.togglestatus on/off` | Enable/disable bot |
| `.mode public/private` | Set who can use commands |
| `.setprefix <prefix>` | Change command prefix |
| `.autorecording on/off` | Show recording presence |
| `.autotyping on/off` | Show typing presence |
| `.anticall on/off` | Auto-reject calls |
| `.welcome on/off` | Welcome new group members |
| `.goodbye on/off` | Goodbye message on leave |
| `.autoread on/off` | Auto-read messages |
| `.autoviewstatus on/off` | Auto-view statuses |
| `.autolikestatus on/off` | Auto-like statuses ❤️ |
| `.antidelete on/off` | Resend deleted messages |
| `.autobio on/off` | Auto-bio toggle |

### 👥 Group
| Command | Description |
|---|---|
| `.requestlist` | List join requests |
| `.acceptall` | Approve all requests |
| `.rejectall` | Reject all requests |
| `.promote @user` | Promote to admin |
| `.demote @user` | Demote from admin |
| `.removeadmins @user` | Remove admin status |
| `.botadmin` | Check if bot is admin |
| `.add <number>` | Add member by number |
| `.addmember <numbers>` | Add multiple members |
| `.tagall` | Mention all members |
| `.hidetag` | Silent mention all |
| `.admincheck` | List group admins |
| `.groupstatus` | Group info and stats |

### 🔨 Admin
| Command | Description |
|---|---|
| `.kick @user` | Remove a member |
| `.kickall confirm` | Remove all non-admins |
| `.end` | Lock group to admins only |
